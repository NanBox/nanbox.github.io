Anroid Wear OS 手表应用开发 - UI

发表于 2019-03-02  |  2019-03-02   |     |   阅读次数

智能手表相对手机来说，由于使用场景不同，且屏幕较小，所以应用的设计、交互和手机是有些区别的。相对来说，布局会更简洁，更多地使用滑动手势进行操作。

为此，在 Wear OS 上，官方提供了一系列新的控件和交互，通过它们，我们可以很方便地打造出一个适合手表交互的应用。
